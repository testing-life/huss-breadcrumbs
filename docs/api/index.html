<huss-breadcrumbs>

Breadcrumbs which can collapse in the middle, leaving only first and several last crumbs visible.

Attributes

NameDescriptionTypeDefault
collapseAfterAfter how many elements should the list collapse, keeping only the first and last three elements (on large screen).number
